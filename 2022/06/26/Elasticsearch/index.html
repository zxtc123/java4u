  • еҜ№ж–ҮжЎЈдёӯзҡ„еҶ…е®№еҲҶиҜҚпјҢеҫ—еҲ°зҡ„иҜҚиҜӯе°ұжҳҜиҜҚжқЎ

д»Җд№ҲжҳҜжӯЈеҗ‘зҙўеј•пјҹ

  • еҹәдәҺж–ҮжЎЈidеҲӣе»әзҙўеј•гҖӮжҹҘиҜўиҜҚжқЎж—¶еҝ…йЎ»е…ҲжүҫеҲ°ж–ҮжЎЈпјҢиҖҢеҗҺеҲӨж–ӯжҳҜеҗҰеҢ…еҗ«иҜҚжқЎ

д»Җд№ҲжҳҜеҖ’жҺ’зҙўеј•пјҹ

  • еҜ№ж–ҮжЎЈеҶ…е®№еҲҶиҜҚпјҢеҜ№иҜҚжқЎеҲӣе»әзҙўеј•пјҢе№¶и®°еҪ•иҜҚжқЎжүҖеңЁж–ҮжЎЈзҡ„дҝЎжҒҜгҖӮжҹҘиҜўж—¶е…Ҳж №жҚ®иҜҚжқЎжҹҘиҜўеҲ°ж–ҮжЎЈid,иҖҢеҗҺиҺ·еҸ–еҲ°ж–ҮжЎЈ

esдёҺmysqlжҰӮеҝөеҜ№жҜ”

mysql elasticsearch иҜҙжҳҺ
table index зҙўеј•е°ұжҳҜж–ҮжЎЈйӣҶеҗҲпјҢзұ»дјјж•°жҚ®еә“зҡ„иЎЁ
row document ж–ҮжЎЈе°ұжҳҜдёҖжқЎжқЎж•°жҚ®пјҢзұ»дјјж•°жҚ®еә“зҡ„иЎҢпјҢж–ҮжЎЈйғҪжҳҜJSONж јејҸ
column field еӯ—ж®өпјҢе°ұжҳҜж–ҮжЎЈдёӯзҡ„еӯ—ж®өпјҢзұ»дјјж•°жҚ®еә“дёӯзҡ„еҲ—
schema mapping жҳ е°„жҳҜзҙўеј•дёӯж–ҮжЎЈзҡ„зәҰжқҹпјҢдҫӢеҰӮеӯ—ж®өзұ»еһӢзәҰжқҹгҖӮзұ»дјјж•°жҚ®еә“зҡ„иЎЁз»“жһ„
SQL DSL DSLжҳҜelasticsearchжҸҗдҫӣзҡ„JSONйЈҺж јзҡ„иҜ·жұӮиҜӯеҸҘпјҢе®һзҺ°CRUD

еҲҶиҜҚеҷЁ

еҲҶиҜҚеҷЁзҡ„дҪңз”ЁжҳҜд»Җд№Ҳпјҹ

  • еҲӣе»әеҖ’жҺ’зҙўеј•ж—¶еҜ№ж–ҮжЎЈеҲҶиҜҚ
  • з”ЁжҲ·жҗңзҙўж—¶пјҢеҜ№иҫ“е…Ҙзҡ„еҶ…е®№еҲҶиҜҚ

IKеҲҶиҜҚеҷЁжңүеҮ з§ҚжЁЎејҸпјҹ

  • ik_smart:жҷәиғҪеҲҮеҲҶпјҢзІ—зІ’еәҰ
  • ik_max_workпјӣжңҖз»ҶеҲҮеҲҶпјҢз»ҶзІ’еәҰ

IKеҲҶиҜҚеҷЁеҰӮдҪ•жү©еұ•иҜҚжқЎпјҹеҰӮдҪ•еҒңз”ЁиҜҚжқЎ

  • еҲ©з”Ёconfigзӣ®еҪ•зҡ„IkAnalyzer.cfg.xmlж–Үд»¶ж·»еҠ жӢ“еұ•иҜҚе…ёе’ҢеҒңз”ЁиҜҚе…ё
  • еңЁиҜҚе…ёдёӯж·»еҠ жӢ“еұ•иҜҚжқЎжҲ–иҖ…еҒңз”ЁиҜҚжқЎ

зҙўеј•еә“ж“ҚдҪң

mappingеұһжҖ§

mappingжҳҜеҜ№зҙўеј•еә“дёӯж–ҮжЎЈзҡ„зәҰжқҹпјҢеёёи§Ғзҡ„mappingеұһжҖ§еҢ…жӢ¬пјҡ

  • type:еӯ—ж®өж•°жҚ®зұ»еһӢпјҢеёёи§Ғз®ҖеҚ•зұ»еһӢпјҡ
    • еӯ—з¬ҰдёІпјҡtext(еҸҜеҲҶиҜҚж–Үжң¬)гҖҒkeyword(зІҫзЎ®еҖјпјҢдҫӢеҰӮпјҡе“ҒзүҢгҖҒеӣҪе®¶гҖҒipең°еқҖ)
    • ж•°еҖјпјҡlong/integer/short/byte/double/float
    • еёғе°”пјҡboolean
    • ж—Ҙжңҹпјҡdate
    • еҜ№иұЎпјҡobject
  • index:жҳҜеҗҰеҲӣе»әзҙўеј•пјҢй»ҳи®Өдёәtrue
  • analyzer:дҪҝз”Ёе“Әз§ҚеҲҶиҜҚеҷЁ
  • properties:иҜҘеӯ—ж®өеӯҗеӯ—ж®ө

зҙўеј•еә“ж“ҚдҪңпјҡ

  • еҲӣе»әзҙўеј•еә“пјҡPUT/зҙўеј•еҗҚ
  • жҹҘиҜўзҙўеј•еә“пјҡGET/зҙўеј•еҗҚ
  • еҲ йҷӨзҙўеј•еә“пјҡDELETE/зҙўеј•еҗҚ
  • ж·»еҠ еӯ—ж®өпјҡPUT/зҙўеј•еҗҚ/_mapping

ж–ҮжЎЈж“ҚдҪңпјҡ

  • ж–°еўһж–ҮжЎЈпјҡPOST/зҙўеј•еә“еҗҚ/_doc/ж–ҮжЎЈid {JSONж–ҮжЎЈ}
  • жҹҘиҜўж–ҮжЎЈпјҡGET/зҙўеј•еә“еҗҚ/_doc/ж–ҮжЎЈid
  • еҲ йҷӨпјҡDELETE/зҙўеј•еә“еҗҚ/_doc/ж–ҮжЎЈid
  • дҝ®ж”№ж–ҮжЎЈпјҡ
    • е…ЁйҮҸдҝ®ж”№пјҢеҲ йҷӨж—§ж–ҮжЎЈпјҢж·»еҠ ж–°ж–ҮжЎЈ PUT/зҙўеј•еә“еҗҚ/_doc/ж–ҮжЎЈid {JSONж–ҮжЎЈ}
    • еўһйҮҸдҝ®ж”№пјҢдҝ®ж”№жҢҮе®ҡеӯ—ж®ө POST/зҙўеј•еә“еҗҚ/_update/ж–ҮжЎЈid {вҖңdocвҖқ:{еӯ—ж®ө}}

ESж”ҜжҢҒ2з§Қең°зҗҶеқҗж Үж•°жҚ®зұ»еһӢпјҡ

  • geo_point:з”ұз»ҸеәҰзә¬еәҰзЎ®е®ҡзҡ„дёҖдёӘзӮ№ вҖң32.8752, 120.2981вҖқ
  • geo_shape:еӨҡдёӘgeo_pointз»„жҲҗзҡ„еӨҚжқӮеҮ дҪ•еӣҫеҪў

еӯ—ж®өжӢ·иҙқеҸҜд»ҘдҪҝз”Ёcopy_toеұһжҖ§е°ҶеҪ“еүҚеӯ—ж®өжӢ·иҙқеҲ°жҢҮе®ҡеӯ—ж®өпјҡ

DSLжҹҘиҜўж–ҮжЎЈ

DSLжҹҘиҜўеҲҶзұ»пјҢеёёи§ҒжҹҘиҜўзұ»еһӢпјҡ

  • жҹҘиҜўжүҖжңүпјҡжҹҘиҜўжүҖжңүж•°жҚ®
  • е…Ёж–ҮжЈҖзҙўпјҡеҲ©з”ЁеҲҶиҜҚеҷЁеҜ№з”ЁжҲ·иҫ“е…ҘеҶ…е®№еҲҶиҜҚпјҢз„¶еҗҺеҺ»еҖ’жҺ’зҙўеј•еә“еҢ№й…Қ
    • match_query
    • multi_match_query
  • зІҫзЎ®жҹҘиҜўпјӣдёҖиҲ¬жҹҘиҜўkeywordгҖҒж•°еҖјгҖҒж—ҘжңҹзӯүпјҢдҫӢеҰӮпјҡ
    • ids
    • range
    • term
  • ең°зҗҶжҹҘиҜўпјҡж №жҚ®з»Ҹзә¬еәҰжҹҘиҜўпјҢдҫӢеҰӮпјҡ
    • geo_distance
    • geo_bounding_box
  • еӨҚеҗҲжҹҘиҜўпјҡеӨҚеҗҲжҹҘиҜўеҸҜд»Ҙе°ҶдёҠиҝ°жҹҘиҜўжқЎд»¶з»„еҗҲиө·жқҘпјҢеҗҲе№¶жҹҘиҜў

е…Ёж–ҮжЈҖзҙў

matchжҹҘиҜў

GET /indexName/_search
{
  "query": {
    "match": {
      "FIELD": "TEXT"
    }
  }
}

multi_match:дёҺmatchжҹҘиҜўзұ»дјјпјҢеҸӘдёҚиҝҮе…Ғи®ёеҗҢж—¶жҹҘиҜўеӨҡдёӘеӯ—ж®ө

GET /indexName/_search
{
  "query": {
    "multi_match": {
      "query": "TEXT",
      "fields": ["FIELD1","FIELD2"]
    }
  }
}

зІҫзЎ®жҹҘиҜў

#termжҹҘиҜў,зІҫзЎ®жҹҘиҜў

{
  "query": {
    "term": {
      "FIELD": {
        "value": "VALUE"
      }
    }
  }
}

#rangeжҹҘиҜў

{
  "query": {

      }
    }
  }
}
